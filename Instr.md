# Команды для работы в git

## Создание пустого репозитория

``````
git init
``````
## Добавление файла к отслеживанию

``````
git add
``````

## Создание коммитов(сохранений)

``````
git commit -m "Messege"
``````

## Список созданных коммитов

``````
git log
``````

git log --oneline
git checkout
